# Icon assets

`icon.ico` is a placeholder generated by `tools/generate_placeholder_icon.py`.
It's a 256×256 32-bit BGRA icon, dark teal background with white "JF" lettering.

To regenerate (if you change the colors or letters in the script):

    .\.venv\Scripts\python.exe tools/generate_placeholder_icon.py

A real designed icon should replace this before any non-beta release. The
JustFixed roadmap tracks this as backlog item B23.
